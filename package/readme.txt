【安装方法】
1、把本文件夹解压到任意目录，例如：D:\crx\package。
2、在Chrome里点击菜单，选择：更多工具→扩展程序。弹出扩展程序(或者地址栏直接输入：chrome://extensions)。
3、打开界面右上方的“开发者模式”开关，此时左上方会出现按钮“加载已解压的扩展程序”。
4、点击后，打开步骤1解压的目录。
5、看到浏览器地址栏右侧出现红色的ymg按钮，安装完成。

【设置方法】

一、打开配置界面

1、右键单击红色按钮。
2、在弹出菜单中点击【选项】，弹出配置页面。

二、站点设置（直接找我帮你也行的）

1、默认配置仅支持“学习强国”、“微信公众号”两类。
2、您可以在“站点设置”新增规则。
3、规则名称最好填写英文、数字和下划线。
4、匹配网站为对网址的正则匹配。
5、选择器请参考jQuery官方文档。
